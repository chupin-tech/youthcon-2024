저도 업무에서 써본 적은 없지만, 이런 기회에 열심히 써보고 싶어서 JDK21 버전을 골랐어요!<br>
혹시 실행 환경에 JDK21 이상 버전이 없으신가요?<br>
없으시다면 아래의 가이드에 따라 JDK21을 설치 및 설정해주세요.

#### JDK21 설치 및 설정 가이드
